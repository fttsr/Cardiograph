# Мобильное приложение для домашнего кардиографа
![photo_2025-08-27_18-05-46](https://github.com/user-attachments/assets/ebf12c3d-673f-4fa8-84d9-ec3ee937f6ef) 
Данное приложение предназначено для работы с устройством - домашним кардиографом. Оно позволяет подключаться к устройству по Bluetooth, получать и обрабатывать данные электрокардиограммы, сохранять результаты измерений и просматривать историю процедур.

## Установка и запуск проекта
### Сервер
npm install  
npm run start
### Мобильное приложение
flutter pub get  
flutter run --release

## Стек технологий 
### Сервер 
- Node.js
- Express.js
- PostgreSQL
- REST API
### Мобильное приложение
- Flutter
- Dart
- BLoC
- Hive
- GetIt
   
Полный список используемых библиотек можно посмотреть в соответствующих файлах зависимостей: *pubspec.yaml* (мобильное приложение) и *package.json* (сервер)  

## Архитектура проекта
### Сервер
Серверная часть реализована на Layered Architecture.
### Мобильное приложение
Мобильное приложение реализовано на Clean Architecture.
